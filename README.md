# Auction

## Installation

Run:
```
yarn
```

## Launch app
1. Start WebSocket server (runs on port `3051`)
```
yarn start-ws-server
```
2. Start web server (runs on port 3050)
```
yarn start-web-server
```
3. Open `http://localhost:3050` URL in Chrome browser (v68+)

## Browser support

* Chrome 68+
